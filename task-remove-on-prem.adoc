= NetApp Console 에서 온프레미스 ONTAP 시스템 제거
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
더 이상 콘솔에서 관리하지 않으려면 온프레미스 ONTAP 시스템을 제거합니다.

.필수 콘솔 역할:
저장소 관리자.link:https://docs.netapp.com/us-en/console-setup-admin/reference-iam-predefined-roles.html["NetApp Console 액세스 역할에 대해 알아보세요."^]

시스템을 제거해도 ONTAP 클러스터에는 영향을 미치지 않습니다.  언제든지 콘솔에서 다시 확인할 수 있습니다.

.단계
. 탐색 메뉴에서 *저장소 > 관리*를 선택합니다.
. *시스템* 페이지에서 온프레미스 ONTAP 시스템을 선택합니다.
. 메뉴 아이콘을 선택하고 *작업 공간에서 제거*를 선택하세요.
. *제거*를 선택하여 확인하세요.
